Sistema de Monitoramento e Controle para Hortas Urbanas
Projeto Técnico - Avaliação 01 (Introdução à Computação - CEUB)
 Visão Geral
Este repositório contém o desenvolvimento técnico de uma solução de infraestrutura de TI voltada para o monitoramento ambiental de pequenas estufas e hortas urbanas. O projeto foi desenvolvido seguindo os requisitos da AV01, focando na integração entre hardware, software e redes para solucionar problemas cotidianos de manutenção botânica em ambientes internos.
 Estudo de Caso: O Problema

    Cenário: Moradores de áreas urbanas com dificuldades em manter a saúde de suas plantas devido à falta de monitoramento constante de umidade e temperatura.
    Lacuna de TI (O Problema): Ausência de um sistema integrado que automatize a rega e forneça dados precisos para o usuário em tempo real.
    Solução: Uma infraestrutura que utiliza sensores para coleta de dados, processamento local e interface remota para gestão inteligente.

 Especificações Técnicas (Arquitetura Robustas)
Para garantir uma arquitetura robusta e justificada, o projeto foi dividido em:
1. Hardware e Redes

    Microcontrolador: ESP32/Arduino (Simulado no Tinkercad).
    Sensores: Sensor de umidade de solo e sensor de temperatura/umidade do ar (DHT11).
    Atuadores: Sistema de bomba d'água para rega automática.
    Rede: Protocolo de comunicação Wi-Fi para envio de métricas.

2. Software e Gestão

    Linguagem: C++ (Arduino IDE) com foco em código limpo e bem identado.
    Sistema de Gestão: Dashboard para visualização de dados históricos e alertas.
    APIs: Integração com APIs climáticas para otimização de recursos hídricos.
